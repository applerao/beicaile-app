## 1. UI 设计规范

### 1.1 色彩系统

#### 主色调

| 色彩名称 | 色值 | 用途 |
|---------|------|------|
| 深空黑 | `#0D0D0D` | 主背景色 |
| 警示红 | `#FF3B30` | 主强调色、CTA 按钮、关键状态 |

#### 辅助色

| 色彩名称 | 色值 | 用途 |
|---------|------|------|
| 碳素灰 | `#1A1A1A` | 卡片背景、次级区域 |
| 金属灰 | `#2D2D2D` | 边框、分割线 |
| 冷白 | `#E5E5E5` | 主文字色 |
| 雾灰 | `#8E8E93` | 次级文字、占位符 |
| 霓虹红 | `#FF6B68` | 悬停状态、渐变辅助 |
| 暗红 | `#9A2A25` | 按下状态、深度阴影 |

#### 状态色

| 状态 | 色值 | 说明 |
|------|------|------|
| 在职 | `#34C759` | 绿色，讽刺意味 |
| 待优化 | `#FF9500` | 橙色，委婉说法 |
| 已毕业 | `#FF3B30` | 红色，直白真相 |
| 冷冻期 | `#5856D6` | 紫色，等待期 |

#### 渐变方案

```css
/* 主渐变 - 深渊 */
background: linear-gradient(180deg, #0D0D0D 0%, #1A1A1A 100%);

/* 强调渐变 - 警示 */
background: linear-gradient(135deg, #FF3B30 0%, #9A2A25 100%);

/* 卡片渐变 - 冷峻 */
background: linear-gradient(145deg, #1A1A1A 0%, #2D2D2D 100%);
```

### 1.2 字体系统

#### 中文字体栈

```css
font-family: 
  'PingFang SC',           /* macOS/iOS 优先 */
```

#### 英文字体栈

```css

  'SF Pro Display',         /* iOS/macOS */
  'Inter',                 /* 跨平台 */
  'Roboto',                /* Android */

```

#### 字号规范

| 层级 | 字号 | 行高 | 字重 | 用途 |
|------|------|------|------|------|
| H1 | 32px | 40px | Bold (700) | 页面标题 |
| H2 | 24px | 32px | Bold (700) | 模块标题 |
| H3 | 20px | 28px | Semibold (600) | 子标题 |
| Body | 16px | 24px | Regular (400) | 正文内容 |
| Caption | 14px | 20px | Regular (400) | 说明文字 |
| Small | 12px | 16px | Regular (400) | 辅助信息 |
| Micro | 10px | 14px | Regular (400) | 标签、角标 |

#### 字重使用

| 字重 | 值 | 使用场景 |
|------|-----|---------|
| Regular | 400 | 正文、说明 |
| Medium | 500 | 强调文字 |
| Semibold | 600 | 子标题、按钮 |
| Bold | 700 | 主标题、关键数据 |

## 3. 品牌视觉元素

### 3.1 Logo 概念

#### 设计理念

> **"在黑暗中寻找光明，在不确定性中寻找确定性"**

Logo 设计融合以下元素：
- **问号的变形** - 代表"被裁了吗"的核心疑问
- **保护盾的轮廓** - 象征互助与保护
- **电路/数据流** - 体现赛博朋克科技感

#### Logo 方案

```
方案 A：问号盾牌
┌─────────────┐
│     ╭───╮   │
│    │  ?  │  │  ← 问号融入盾牌轮廓
│     ╰───╯   │
│   #FF3B30   │
└─────────────┘

方案 B：断裂链条
┌─────────────┐
│  ━━⚡━━     │  ← 断裂的链条，象征被裁
│   #FF3B30   │
└─────────────┘

方案 C：极简文字
┌─────────────┐
│  被裁了吗？  │  ← 纯文字 Logo
│   #FF3B30   │
└─────────────┘
```

#### 推荐使用

**方案 A（问号盾牌）** 作为主 Logo

#### Logo 使用规范

```
最小尺寸：24px × 24px
安全边距：Logo 高度的 1/4
背景：深空黑 #0D0D0D 或纯白
禁忌：不得拉伸变形、不得改变颜色
```

### 3.2 图标风格

#### 设计原则

- **线性图标** - 1.5px 描边，简洁利落
- **圆角端点** - 2px 圆角，柔和但不失冷峻
- **双色系统** - 主色冷白 + 强调色警示红

#### 图标规范

```css
.icon {
  width: 24px;
  height: 24px;
  stroke: #E5E5E5;
  stroke-width: 1.5px;
  fill: none;
  stroke-linecap: round;
  stroke-linejoin: round;
}

.icon-active {
  stroke: #FF3B30;
}
```

#### 核心图标集

| 图标 | 名称 | 使用场景 |
|------|------|---------|

## 4. 交互动效说明

### 4.1 动效原则

> **"克制、精准、有态度"**

| 原则 | 说明 |
|------|------|
| 目的性 | 每个动效都有明确的功能目的 |
| 克制 | 不过度使用，避免干扰 |
| 一致 | 全应用动效语言统一 |
| 性能 | 60fps 流畅，不影响性能 |

### 4.2 基础动效曲线

```css
/* 标准缓动 - 大多数场景 */
--ease-standard: cubic-bezier(0.25, 0.1, 0.25, 1);

/* 加速缓动 - 进入动画 */
--ease-in: cubic-bezier(0.4, 0.0, 1, 1);

/* 减速缓动 - 退出动画 */
--ease-out: cubic-bezier(0.0, 0.0, 0.2, 1);

/* 弹性缓动 - 强调反馈 */
--ease-bounce: cubic-bezier(0.68, -0.55, 0.265, 1.55);
```

### 4.3 核心动效

#### 签到按钮动画

```
状态流程：
[默认] → [按下] → [释放] → [成功] → [完成]

动画细节：
1. 按下：按钮缩小至 95%，0.1s
2. 释放：按钮恢复 100%，0.1s
3. 成功：红色粒子爆发，0.3s
4. 完成：震动反馈 + 数字滚动

粒子效果：
    ·
  · · · ·
· · ★ · ·   ← 从按钮中心爆发
  · · · ·
    ·
```

**代码示例：**
```css
@keyframes checkin-burst {
  0% {
    transform: scale(0);
    opacity: 1;
  }
  100% {
    transform: scale(2);
    opacity: 0;
  }
}

.particle {
  animation: checkin-burst 0.3s var(--ease-out);
}
```

#### 页面切换动画

```
Tab 切换：
[当前页] →→→ [新页面]
    滑出左      滑入右

时长：0.25s
曲线：var(--ease-standard)
```

**代码示例：**
```css
.page-enter {
  transform: translateX(100%);
  animation: slide-in 0.25s var(--ease-standard);
}

.page-exit {
  animation: slide-out 0.25s var(--ease-standard);
}

@keyframes slide-in {
  from { transform: translateX(100%); }
  to { transform: translateX(0); }
}

@keyframes slide-out {
  from { transform: translateX(0); }
  to { transform: translateX(-100%); }
}
```

#### 卡片展开动画

```
[折叠状态]
┌─────────────┐
│ 卡片标题  ▶ │
└─────────────┘
       ↓
[展开状态]
┌─────────────┐
│ 卡片标题  ▲ │
├─────────────┤
│             │
│  详细内容   │
│             │
└─────────────┘

时长：0.2s
曲线：var(--ease-out)
```

#### 加载动画

```
旋转问号：
   ?
  / \      ← 问号绕中心旋转
 ·   ·

时长：1s 循环
曲线：linear
```

**代码示例：**
```css
@keyframes spin {
  from { transform: rotate(0deg); }
  to { transform: rotate(360deg); }
}

.loading {
  animation: spin 1s linear infinite;
}
```

#### 成功状态动画

```
对勾绘制：
    ╭─
   │  ✓
    ╰

时长：0.4s
曲线：var(--ease-bounce)
```

### 4.4 微交互反馈

#### 震动反馈

| 场景 | 震动模式 |
|------|---------|
| 签到成功 | 重震动 (HapticFeedback.heavy) |

#### 声音反馈

| 场景 | 音效 |
|------|------|

### 4.5 转场特效

#### 故障转场 (特殊场景)

```
效果描述：

时长：0.5s
使用频率：极低 (仅关键状态变化)
```
